User Giriş Paneli Nedir?

Mail System sayfasının sağ üst köşesinden Giriş Yap kısmından giriş yapılır. Bu panele giren kişi mail alma ve gönderme işlemi yapabilme yetkisine sahip olmuştur. Eğer hoca ek özellik olarak mailleri onaylayabilir.
